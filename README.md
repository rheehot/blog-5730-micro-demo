# NX 환경 준비
- Angular v9.* 
- @angular/cli v9.1.6
- RxJS v6.5.*
- Typescript v3.8.*
- NodeJS v12.16.*
Node Version Manager(nvm)를 통해 로컬환경에 여러 Node버전을 관리하자.

## NodeJS
```
$ nvm install 12.16.2
$ nvm alias default 12.16.2
$ nvm use 12.16.2
```

## Angular/CLI 최신버전 사용
```
$ npm i -g @angular/cli@latest
$ npm i -g @nrwl/cli@latest
$ npm i -g yarn@latest
```

## Install packages of package.json
```
$ yarn add
```
